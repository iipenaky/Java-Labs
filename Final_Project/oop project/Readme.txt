Description:
MentorConnect is a Java application that facilitates mentor-mentee matching.

============================================================
Requirements:
- Java Development Kit (JDK) installed
- A Java IDE or text editor for code editing.

============================================================
Installation:
1. Ensure that you have JDK installed on your system.

======================================================
Usage:
1. Open a terminal or command prompt.
2. Navigate to the directory where MentorConnect’s files are located.

To compile the program, use the following command:

$ javac Main.java

To run the program, use the following command:

$ java Main

Follow the on-screen instructions to proceed with the mentor or mentee's actions.
